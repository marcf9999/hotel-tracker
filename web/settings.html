How It Works

The checker runs every 2 hours via GitHub Actions. It visits each hotel's Marriott calendar page with an undetected Chrome browser, parses per-night availability and pricing, and stores results in the database.

You'll receive an immediate email when any tracked hotel's dates open up. If bot detection blocks the checker, you'll get a notification about that too (max once per day).
